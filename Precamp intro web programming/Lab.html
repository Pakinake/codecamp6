1.Attribute

Tag "h"

ใช้สำหรับเน้นคำที่เป็นหัวข้อเรื่อง โดยมี h1-h6 ขนาดตัวอักษรจะไล่จากใหญ่ที่สุดไปจนเล็กที่สุด ไม่มี Attribute เสริมเป็นของตัวเอง


Tag "p"

เป็น Tag paragraph ใช้สำหรับพิมพ์เนื้อหาลงไป ไม่มี attribute เสริมเป็นของตัวเอง


Tag "div"

ใช้พิมพ์ข้อความเหมือน Tag p แต่จะเว้นบรรทัดลงมาเป็นบรรทัดใหม่ ไม่มี attribute เสริม


Tag "img"

  • alt ใช้อธิบายภาพที่เราแสดงไว้ในกรณีหากภาพไม่สามารถแสดงได้ เพื่อให้กลุ่มผู้มีปัญหาทางประสาทสัมผัสรับรู้รุปภาาพนั้นได้
  • height ปรับขนาดความสูงของภาพ
  • width ปรับความกว้างของภาพ
  • src กำหนดแหล่งที่มาของภาพว่านำภาพมาจากที่ไหน

Tag "a"

  • download เริ่มการดาวน์โหลดเมื่อคลิกลิ้งก์
  • href ระบุ URL ที่จะทำการลิ้งก์ไป
  • target ระบุวิธีการเปิดหน้าเพจ

Tag "ul"

ใช้ระบุข้อความแบบเป็นลิสโดยหน้าข้อความจะเป็นสัญลักษณ์ ไม่มี attribute เสริมเป็นของตัวเอง

Tag "ol"

  • reversed กลับลำดับจากเริ่มต้นที่ 1 เป็นเริ่มจากตัวเลขที่มากที่สุดตามจำนวนลิส
  • start กำหนดตัวเลขเริ่มต้นที่จะใช้เป็นลิสเช่น เริ่มที่ 50 เป็นต้นไป
  • type ระบุประเภทของลิสที่จะใช้เช่นจากปกติจะเป็น 1 2 3 ให้เปลี่ยนเป็นตัวอักษร a b c

Tag "table"

ใช้สร้างตารางขึ้นมา ไม่มี attribute เสริมเป็นของตัวเอง

Tag "form"

  • accept-charset กำหนดรูปแบบcharacter encodings ที่จะส่งข้อมูล
  • action กำหนด URL ที่จะส่งข้อมูลไป
  • autocomplete เติมคำอัตโนมัติ

Tag "button"

  • autofocus เน้นปุ่มนั้นให้น่าสนใจ เช่นมีเส้นสีฟ้าขึ้นมาเน้นรอบปุ่มเมื่อทำการโหลดเพจขึ้นมา
  • disabled ยกเลิกการทำงานของปุ่มๆนั้น
  • name ตั้งชื่อให้ปุ่มนั้นๆ
  • type เลือกTypeให้ปุ่มนั้นๆ

Ref : W3School.com


2.Differentiation

Tag Similar Different
hr&br เว้นวรรคบรรทัดเหมือนกัน brจะเว้นบรรทัดอย่างเดียว ส่วนhrจะมีเส้นขึ้นมากั้นระหว่างบรรทัด
ul&ol สร้างลำดับเป็นlistเหมือนกัน ulจะกำหนดหน้าlistเป็นสัญลักษณ์ ส่วนolจะกำหนดเป็นตัวเลขหรือตัวอักษร
select&checkbox&radio botton สร้างตัวเลือกขึ้นมาให้เลือกเหมือนๆกัน select จะสร้างเป็นdropboxขึ้นมาให้เลือกเพียง 1 ตัวเลือก
checkboxจะสร้างตัวเลือกขึ้นมามีช่องสี่เหลี่ยมด้านหน้าสามารถเลือกตัวเลือกได้ทั้งหมด
radio button จะสร้างตัวเลือกขึ้นมาด้านหน้ามีปุ่มเป็นวงกลมเลือกตัวเลือกได้ 1 ตัวเลือก
div&span สามารถกำหนด style ของข้อความที่อยู่ใน Tag ได้เหมือนๆกัน divจะแยกข้อความในแต่ละtagออกขึ้นเป็นบรรทัดใหม่ spanจะรวบข้อความในtagมาอยู่บรรทัดเดียวกัน
script&style ใช้แสดงผลลัพธ์ออกมาบนBrowserเหมือนๆกัน script จะเกี่ยวกับการจัดการหรือความควบคุมข้อมูลต่างๆที่จะแสดงออกมา
style จะแสดงลักษณะต่างๆเช่นสีของตัวอักษร ขนาดฟอร์น

3.Input type

Input type การใช้งาน
button สร้างปุ่มที่สามารถคลิกได้ขึ้นมา
checkbox กำหนดตัวเลือกขึ้นมีกล่องสี่เหลี่ยมให้เลือกด้านหน้า สามารถเลือกตัวใดตัวหนึ่งหรือทั้งหมดได้
color สร้างกล่องสีขึ้นมาให้เลือกสีที่ต้องการ
date สร้างกล่องที่สามารถเลือกวัน-เดือน-ปีขึ้นมา
datetime-local สร้างกล่องขึ้นมาคล้ายกับdateแต่จะมีส่วนของเวลาเพิ่มขึ้นมา
email สร้างช่องให้กรอกemail
file เรียกหน้าต่างเลือกไฟล์สำหรับuploadขึ้นมา
hidden ซ่อนdataที่ทำการsubmitไปจากuser
image ใช้รูปภาพแทนปุ่มsubmit
month สร้างกล่องเหมือนdateแต่จะแสดงแค่เดือน-ปี
number สร้างกล่องที่สามารถระบุตัวเลขขึ้นมา สามารถกำหนดช่วงตัวเลขที่จะให้ใส่ได้
password สร้างกล่องใส่Password
radio กำหนดตัวเลือกขึ้นมาโดยด้านหน้าจะเป็นปุ่มวงกลมให้เลือก เลือกได้เพียง 1 ตัวเลือก
range สร้างปุ่มที่สามารถเลื่อนซ้ายขวาได้และสามารถระบุช่วงค่าmin-maxได้
reset สร้างปุ่มที่กดแล้วจะกลับไปค่าเริ่มต้นได้
search สร้างกล่องข้อความสำหรับใส่สิ่งที่ต้องการจะค้นหา
submit สร้างปุ่มsubmitสำหรับส่งข้อมูล
tel สร้างกล่องสำหรับใส่เบอร์โทร สามารถกำหนดรูปแบบการใส่ได้
text สร้างกล่องใส่ข้อความตัวอักษร
time สร้างกล่องที่สามารถระบุเวลาได้
url สร้างกล่องที่สามารถใส่ลิ้งก์ขึ้นมา
week สร้างกล่องขึ้นมาลักษณะคล้ายdateแต่จะระบุวันสัปดาห์ที่เท่าไหร่ของปี
