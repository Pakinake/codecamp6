Property Value unit Different
คือส่วนของคุณสมบัติต่างๆที่เราต้องการให้แสดงออกมาใน Html เช่น สี,ขนาด,กรอบ, ใน Precamp สอนไว้ 5 ประเภทืคือ
  1. for content
  2. for border
  3. for padding
  4. for margin
  5. for element
 คือค่าต่างๆที่เราต้องการให้ Property ที่เราเลือกแสดงออกมา เช่น ลักษณะตัวอักษร,ขนาดของตัว,สีที่ต้องการ,รูปแบบของกรอบ unit มี 2 ประเภท คือ Absolute และ Relative
  • Absolute จะกำหนดขนาดเป็นแบบ cm,mm,px เป็นการกำหนดแบบตายตัวไม่เกี่ยวข้องกับ element อื่นๆ
  • Relative จะกำหนดแบบมีความสัมพันธ์กับ element หรือขนาดจอ ให้เป็นอัตราส่วนเท่าไหร่ต่อ element นั้นๆหรือขนาดของจอแสดงผล
 Property จะกำหนดสิ่งที่เราต้องการให้คุณสมบัตินั้นแสดงออกมา ส่วนValue จะกำหนดค่าว่า Property นั้นจะแสดงออกมาขนาดเท่าไหร่ สีอะไร
