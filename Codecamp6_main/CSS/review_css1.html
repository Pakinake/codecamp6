คือส่วนประกอบของคำสั่งเพื่อให้เกิดผลตามที่เราต้องการประกอบด้วย Selector, Property, Value มีการเขียนได้ 3 แบบคือ External, Internal, Inline มีทั้งหมด 5 แบบ Simple selectors, Combinator selectors, Pseudo-class selectors, Pseudo-elements selectors, Attribute selectors มี 4 แบบ descendant selector (space), child selector (>), adjacent sibling selector (+), general sibling selector (~)

EX Syntax

table { border-collapse: collapse; }


EX Writing Style

  • External จะทำการสร้างไฟล์แยกออกมาจาก html โดยสกุลไฟล์จะลงท้ายด้วย.css แล้วนำเข้าไปในhtmlด้วย Tag link
  • Internal ใช้ Tag Style ภายใน html แล้วทำการกำหนดค่าต่างๆจากภายใน HTML ได้เลย เหมาะกับการตกแต่งไม่เยอะมาก
  • Inline คือการแทรก Systax ไว้ใน Tag HTML เลย เช่น div style="font-size: 10px;"


EX Selector

  • sample selector: div { font-size: 30px; }
  • Pseudo-class: a:hover { color: red; }
  • Pseudo-elements: p::first-line { color: red; }
  • Attribute selectors: [title~="flower"] { border: 5px solid yellow; }


EX Combinator

div ~ p { background-color: yellow; }
